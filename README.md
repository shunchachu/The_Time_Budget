# ⏳ Relic (時光餘額) - Web App MVP

> 「如果生命有期限，我們還能見幾次面？」

Relic 是一款極簡、專注於情感體驗的 Web App。透過將抽象的「時間」轉換為具象的「剩餘相處次數」，喚醒現代人對陪伴親友的重視，並透過互動設計鼓勵實際行動。

## ✨ 核心解決痛點 (The "Why")
在快節奏的現代社會，我們常覺得「以後還有時間」，從而疏於聯繫重要的人。
**UX 洞察：** 數字比文字更具衝擊力。當使用者看到「這輩子只剩 40 次見面機會」時，這種由抽象轉為具象的**視覺衝擊 (Visual Impact)**，能有效打破使用者的拖延心理。

## 🚀 專案亮點與功能 (Features)
- **視覺心理學：** 採用日系 Wabi-sabi (物哀) 美學與毛玻璃質感 (Glassmorphism)，營造平靜卻帶有反思空間的視覺氛圍。
- **互動設計 (微交互)：** 透過 Vanilla JS 實作順滑的「數字跳動計算」動畫，增強數據呈現的重量感。
- **行動解鎖 (Behavioral Nudge)：** 最具巧思的「+1」按鈕機制。不僅是計算，更鼓勵使用者點擊「安排驚喜見面」，給予正向的文字回饋與微動畫特效，將焦慮轉化為行動。
- **Share-ready 設計：** 結果頁面採用電影台詞般的排版比例，方便使用者截圖分享至 Instagram Stories。

## 🛠️ 技術選型 (Tech Stack)
- **HTML5** (語意化標籤)
- **CSS3** (CSS Variables, Flexbox, Keyframe Animations, Glassmorphism 濾鏡)
- **Vanilla JavaScript (ES6)** (客製化 requestAnimationFrame 數字動畫)

## 💻 本地執行方式 (How to run)
1. Clone 此專案至本地端。
2. 直接點擊開啟 `index.html` 即可在瀏覽器中體驗完整功能。
